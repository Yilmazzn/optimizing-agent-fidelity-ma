## Toggle stars without opening messages
1. Click the target folder in the left **Folder Pane**.
2. In the message/thread list, find the **Star** column (hollow/filled star per row).
3. Click the star for a message to toggle it (hollow ⇄ filled).

## If the Star column isn’t visible
- Widen the thread list / columns.
- Use the column chooser in the header row:
  - Click the small **column selector** button at the far right of the column headers (or right‑click the header row).
  - Enable **Star** (may be labeled **Starred**).
