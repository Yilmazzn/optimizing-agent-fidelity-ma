## Play an HLS/DASH manifest URL in VLC
1. Open **VLC**.
2. Go to **Media → Open Network Stream…** (often **Ctrl+N** on Windows/Linux).
3. Paste the link into **Network URL**.
4. Click **Play** (or press **Enter** if the URL field is focused).
5. If the player is blank at first, wait ~**5–15 seconds** for buffering/initialization before retrying or assuming it failed.
