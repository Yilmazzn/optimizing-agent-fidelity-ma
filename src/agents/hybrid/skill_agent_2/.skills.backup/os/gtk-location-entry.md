## Jump to a directory in a GTK file chooser
1. In an Open/Save/Export dialog, press **Ctrl+L** to focus the **Location** (path) field.
2. Type/paste a path (e.g., `~/Desktop` or `/home/<user>/Desktop`).
3. Press **Enter** to navigate.
