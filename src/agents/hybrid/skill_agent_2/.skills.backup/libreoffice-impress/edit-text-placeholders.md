## Edit text in a slide placeholder/text box
1. In the left **Slides** pane, click the target **slide thumbnail**.
2. On the slide canvas, click **inside** the placeholder—ideally on the prompt text (e.g., **“Click to add Text”**).
3. If you only select the frame (no caret): **double‑click** inside, or press **F2**.

## Add/edit speaker notes (Notes view)
1. Select the slide thumbnail in the **Slides** pane.
2. Go to **View ▸ Notes**.
3. If you don’t see the notes body: the notes placeholder is **below the slide image** on the page—use **Page Down** (or scroll) until you see the large blank **“Click to add Notes”** region.
4. **Double‑click inside that blank notes region** to ensure the caret is in the notes body.

### Replace the entire note text (avoids finicky single-character edits)
1. With the caret in the notes body, press **Ctrl+A**.
2. Paste/type the full intended notes text.
3. Press **Esc** to leave text-edit mode.

## Exit text edit mode (so you can move the object)
- Press **Esc** (often **twice**) until the **text caret disappears** and you see the object’s **selection frame/handles**.

## Reposition a placeholder/text box (dragging doesn’t move it)
1. Ensure you’re **not** in text edit mode (caret not visible).
2. Click the **object border/frame** (not inside the text).
3. Drag from the border/handles to reposition.

## Precise placement (no dragging)
- **View ▸ Sidebar ▸ Properties ▸ Position and Size** → set X/Y.

**Common pitfall:** If you can type/edit text, you’re still in text-edit mode—dragging will select text instead of moving the object.
