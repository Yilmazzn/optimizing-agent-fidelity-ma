## Subscript/superscript a single character (e.g., the “2” in H2O)
1. Enter text edit mode (**F2** or double‑click until the caret is visible).
2. Place the caret **after** the target character.
3. Select only that character: **Shift+Left** (or Shift+Right).
4. Toggle:
   - **Subscript:** **Ctrl+Shift+B**
   - **Superscript:** **Ctrl+Shift+P**
5. Press an **arrow key** (or click elsewhere) to clear selection so you don’t keep typing in sub/superscript.

## Menu fallback (most reliable)
1. Select the text while in text edit mode.
2. **Format ▸ Character… ▸ Position** tab → **Subscript** / **Superscript** → **OK**.

Tip: If shortcuts affect the whole text box (or do nothing), you’re not in text edit mode—press **F2** and try again.
