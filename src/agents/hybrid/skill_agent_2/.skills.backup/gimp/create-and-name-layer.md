## Create a new named layer (no Layers dock needed)
1. Go to **Layer > New Layer…** (shortcut **Ctrl+Shift+N**).
2. In the dialog, set **Name** (e.g., `Square`).
3. Click **OK** (or press **Enter**) to create the layer.
