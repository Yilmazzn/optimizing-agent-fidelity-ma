## Set undo history depth
1. Go to **Edit > Preferences**.
2. In the left tree, expand **System Resources**.
3. Set **Minimal number of undo levels** to the value you want.
4. Click **OK** to apply.

Note: This setting is in Preferences (not the Edit/Undo menu).
